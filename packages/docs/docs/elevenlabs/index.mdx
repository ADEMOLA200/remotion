The `@remotion/elevenlabs` package provides utilities for converting the output of ElevenLabs APIs into an array of [`Caption`](/docs/captions/caption) objects, making them compatible with other caption-related Remotion APIs.

It supports converting the output of both the [Speech to Text API](https://elevenlabs.io/docs/api-reference/speech-to-text/convert) and the [Text to Speech with timestamps API](https://elevenlabs.io/docs/api-reference/text-to-speech-with-timestamps).
