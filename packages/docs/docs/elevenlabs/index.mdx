The `@remotion/elevenlabs` package provides utilities for working with the ElevenLabs API.

It provides a transformation function for converting the output of the ElevenLabs API into an array of [`Caption`](/docs/captions/caption) objects in order to make the captions compatible with other caption-related Remotion APIs.
